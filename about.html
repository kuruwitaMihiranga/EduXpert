About the Product

This project aims to develop a software system that might enhance the effectiveness of distant learning by removing the question areas. Using an iris tracking method, the program has a study tracker capable of capturing students’ behavior during class hours. A dynamic question generator is designed to aid students through the process of learning by them. A viva automation tool that can replace the online viva examination procedure via speech recognition and a document simplifier that will help in lexical simplification using a web scraper and reinforcementlearning unit. Existing shortcomings in current learning management systems will be addressed by a revolutionary system with these components.
